## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/megh-0610/meghana-portfolio.git
```

Move into the project

```bash
cd meghana-portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
